# 訳文工房 (Yakubun Kobo)

![Build Status](https://img.shields.io/badge/build-passing-brightgreen)
![Version](https://img.shields.io/github/v/release/suzuryuquark/Yakubun_Kobo_Project)
![License](https://img.shields.io/badge/license-Proprietary-red)

**訳文工房 (Yakubun Kobo)** は、Googleの強力なAIモデル「Gemini」を搭載した、Windows向けの多機能翻訳アプリケーションです。PDFファイルの**AIによる高度な解析**、長文を快適に扱うための**ページング機能**、専門用語を正確に反映させる**ユーザー辞書**、訳文に解説を自動で加える**注釈機能**など、高度な機能で翻訳作業を強力にサポートします。

## 主な機能

*   **高精度なAI翻訳**: Googleの最新モデル「Gemini」を活用。大規模な文脈を理解した自然な翻訳を提供し、敬体（です・ます）と常体（だ・である）の切り替えも自在です。
*   **多彩なファイル形式の読み込み**: テキスト、PDFに加え、Word (.docx)、OpenDocument (.odt)、Markdown (.md)、HTML (.html) を直接読み込み可能。
*   **AIによる高度なファイル解析**: 複雑なレイアウトのPDFも、AIが構造（見出し、リスト、脚注など）を理解して正確にテキストを抽出。元の書式を最大限維持できます。
*   **科学技術文書への対応**: 原文中の数式を検出し、MathML形式に変換して美しく表示。数式を翻訳対象から除外することで、専門文書の品質を向上させます。
*   **大規模文書対応のページング**: 数百ページ（最大500ページ）の長文も自動分割。分割・結合・入れ替えや、特定ページへの高速ジャンプも自由自在です。
*   **継続的な翻訳の改善**: 生成された訳文に対し、「もっと自然に」「この用語を使って」といった追加指示を与えて、AIと共に訳文を洗練できます。
*   **コンテキストと特記事項の自動生成**: 文書の背景情報（タイトル、著者等）から、最適な翻訳指示（特記事項）をAIが自動生成。翻訳の精度を劇的に向上させます。
*   **精度を高めるユーザー辞書と表記揺れ統一**: 専門用語を登録して一貫性を確保。プロジェクト全体をスキャンして表記揺れを自動検出し、一括修正する機能も搭載。
*   **深い理解を助ける注釈・語句解説**: 難解な表現への自動注釈付与や、選択した語句の語源・類義語・例文まで含めた詳細なAI解説機能。
*   **多様な形式へのエクスポート**: 翻訳結果を書式付きPDF、EPUB、HTML、プレーンテキストとして出力。EPUBでは表示画像を指定することもできます。
*   **プロジェクト管理と自動保存**: 作業内容を丸ごと専用形式 (.tapp) で保存。万が一に備えた自動保存機能や、作業中断箇所の記憶で安心して作業に集中できます。
*   **柔軟なAPI管理と快適な操作性**: 最大10個のAPIキーを使い分け可能。ウィンドウ配置の記憶やIME入力の安定化など、プロの道具としての細やかな設計。

## インストール方法

### 1. インストーラーのダウンロード

1.  リポジトリの **Releasesページ** にアクセスします。
2.  最新バージョンのアセット（Assets）から `YakubunKobo_Setup-x.x.x.exe` という名前のファイルをダウンロードします。

> **Warning**
> アプリケーションにはデジタル署名が行われていないため、ブラウザやWindowsによってダウンロード時や実行時に警告が表示されることがあります。これはファイルが危険であることを意味するものではありません。
>
> 詳しい対処法については、[**ダウンロードとインストールの手順**](download-guide.md)をご覧ください。

### 2. インストールの実行

1.  ダウンロードした `.exe` ファイルを実行します。
2.  画面の指示に従ってインストールを進めてください。

> **Note**
> インストーラーは、アプリケーションの実行に必要な以下の環境がPCにインストールされているか自動で確認します。もしインストールされていない場合は、インストールを促すメッセージが表示されます。
> *   **.NET 8 Desktop Runtime (x64)**
> *   **Microsoft Edge WebView2 Runtime**

## 使い方

### 1. APIキーの設定

初めてアプリケーションを起動すると、APIキーの設定を求められます。

1.  **Google AI Studio** にアクセスし、Googleアカウントでログインします。
2.  `Create API key` ボタンをクリックして、新しいAPIキーを生成します。
3.  生成されたAPIキーをコピーし、アプリケーションの入力ダイアログに貼り付けて保存してください。

**複数のAPIキーの管理**:
*   メニューバーの `設定` > `APIキーを設定` から、最大10個のAPIキーをラベル付きで登録できます。
*   メインウィンドウ上部のドロップダウンから、使用するキーを簡単に切り替えられます。

### 2. 基本的な翻訳

1.  左側の「原文」ペインに翻訳したいテキストを入力または貼り付けます。
2.  原文の言語（自動検出も可能）と、翻訳先の言語を選択します。
3.  必要に応じて専門分野（ドメイン）や文体を指定します。
4.  `翻訳` ボタンをクリックすると、右側の「訳文」ペインに結果が表示されます。

> **Note**
> 翻訳方式は、原文を丸ごとAPIに渡してストリーミング形式で訳文を受け取るため、文脈を保った自然で流暢な訳文が生成されます。翻訳完了後、必要に応じて「翻訳の改善」機能で訳文を洗練したり、ツールメニューから「相互ハイライトの付与」や「注釈を生成」を実行したりできます。

### 3. ユーザー辞書

メニューバーの `ユーザー辞書` > `辞書を編集` から、辞書登録ウィンドウを開くことができます。ここで単語やフレーズを登録しておくと、翻訳時に自動で適用されます。

### 4. ファイル読み込み

*   **原文ファイル**: 直接入力のほか、テキスト、PDF、Word、OpenDocument、Markdown、HTMLファイルをメニューバーの `ファイル` > `原文をファイルから読み込む` で読み込めます。
*   **AIファイル解析**: PDFファイルの読み込みには、高度なAI解析機能を利用できます。この機能を有効にするには、メニューバーの `ツール` > `オプション` から「AIによるファイル解析を有効にする」をオンにしてください。
    *   **利点**: 複雑なレイアウトや複数カラムの文書も高精度で抽出できます。
    *   **注意**: ファイル内容がGoogle Gemini APIに送信されます。機密文書の場合は無効にしてローカル解析を使用してください。

### 5. 各種エディタ機能

*   **原文コンテキストと特記事項**: 画面上部の入力欄に文書の背景情報を入力することで、翻訳の精度が向上します。「AIで生成」ボタンを押すと、これらの情報から翻訳の指示をAIが自動で作成します。
*   **翻訳の改善**: メニューバーの `ツール` > `翻訳を改善...` から、既存の訳文をベースに「もっと自然に」「簡潔に」などの追加指示を与えて改善を実行できます。
*   **相互ハイライト**: メニューバーの `ツール` > `相互ハイライトの付与` で、原文と訳文の対応箇所を強調表示します。
*   **数式表示**: 原文中の数式表現は自動的にレンダリングされ、数式を除外した正確な翻訳が行われます。
*   **表記揺れの統一**: メインメニューの `ツール` > `表記揺れの統一` から、ドキュメント全体の表記のばらつきをスキャン・一括置換できます。
*   **注釈と解説**: 訳文ペイン上部の「注釈を生成」ボタンや、右クリックメニューの「語句の解説」から、AIによる詳細な説明を確認できます。

### 6. ファイルのエクスポート

翻訳が完了したら、訳文を高品質なファイルとして出力できます。

1.  メニューバーの `ファイル` > `訳文をエクスポート` を選択。
2.  エクスポート形式（PDF / EPUB / HTML / テキスト）を選択。
3.  オプション（注釈の配置、目次生成、カバー画像など）を設定して出力。

## 注意事項

*   **機密情報とプライバシーについて**:
    *   本ソフトウェアは、翻訳処理のために原文テキストを外部のAPI（Google Gemini API）に送信します。
    *   **AIファイル解析機能**を有効にした場合、PDF文書の内容もGemini APIに送信されます。この機能はデフォルトで無効になっており、オプション設定で明示的に有効化する必要があります。
    *   個人情報や企業秘密など、機密性の高い情報を含む文書は翻訳またはアップロードしないでください。
    *   本ソフトウェアがユーザーの個人情報を意図的に収集・保存することはありませんが、APIの利用規約も併せてご確認ください。詳細はヘルプメニューの「プライバシーポリシー」をご覧ください。
*   **生成物の正確性について**: 本ソフトウェアが生成する訳文・注釈・語句説明・ファイル解析結果の正確性は保証されません。これらの情報を利用した結果生じたいかなる不利益についても、開発者は責任を負いません。詳細は利用規約をご確認ください。

## 謝辞

このアプリケーションは、以下の素晴らしいライブラリやツールの上に成り立っています。

### フレームワーク・ランタイム
*   .NET 8
*   WPF (Windows Presentation Foundation)
*   Microsoft.Extensions.Configuration
*   Microsoft.Extensions.DependencyInjection

### UI・ブラウザ統合
*   Microsoft WebView2

### AI・API統合
*   Google.GenerativeAI (Gemini API)

### ドキュメント処理
*   **PDF処理**: PdfPig（テキスト抽出）、QuestPDF（出力生成）
*   **EPUB処理**: QuickEPUB（出力生成）、VersOne.Epub（読み込み/検証）
*   **Word文書**: DocX
*   **OpenDocument**: AODLCore

### HTML・Markdown処理
*   Markdig（Markdown→HTML変換）
*   AngleSharp（HTML DOM操作）
*   HtmlAgilityPack（HTML処理）

### 画像処理
*   SixLabors.ImageSharp（カバー画像生成）

### ビルド・デプロイ
*   Inno Setup（インストーラー作成）
